# Unit 1 — Nanoscale Modeling

## Overview

Introduction to computational simulation at the nanoscale using the Atomic Simulation Environment (ASE).

## Topics

- Fundamentals of molecular simulation
- Gold nanoparticle structure and optimization
- Structural analysis: RDF and coordination numbers
- Visualization of 3D atomic structures

## Key Technologies

- **ASE** — Atomic Simulation Environment
- **NumPy / SciPy** — Numerical computation
- **Matplotlib** — Scientific visualization

## Learning Outcomes

After completing this unit you will be able to:

1. Build and optimize atomic structures in Python
2. Calculate radial distribution functions (RDF)
3. Analyze coordination numbers in nanoparticles
4. Visualize 3D structures interactively
