# Antigravity Nano Research — Multiagentic Core

> **Curso de IA Aplicada à Pesquisa Científica e Tecnológica — Nanotecnologia**

Curso estruturado completo (Unidades 1–6) para aprender IA aplicada à nanotecnologia — desde simulação molecular até sistemas multi-agente e implantação de modelos em produção.

---

## O que é isto?

Este repositório combina dois campos raramente ensinados juntos:

- **Nanotecnologia Computacional**: dinâmica molecular (ASE), DFT, otimização de nanopartículas, análise XRD
- **Sistemas de IA Modernos**: LLMs, frameworks multi-agente (Google ADK 1.0, CrewAI, LangChain), protocolo A2A, RAG, implantação em produção

## Início Rápido

```bash

jupyter lab
```
