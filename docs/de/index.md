# Antigravity Nano Research — Multiagentischer Kern

> **KI Angewandt auf Wissenschaftliche und Technologische Forschung — Nanotechnologie**

Vollständiger strukturierter Kurs (Einheiten 1–6) zum Erlernen von KI angewandt auf Nanotechnologie — von der Molekularsimulation bis hin zu Multi-Agenten-Systemen und Modell-Deployment in der Produktion.

---

## Was ist das?

Dieses Repository kombiniert zwei Bereiche, die selten zusammen gelehrt werden:

- **Computational Nanotechnologie**: Molekulardynamik (ASE), DFT, Nanopartikel-Optimierung, XRD-Analyse
- **Moderne KI-Systeme**: LLMs, Multi-Agenten-Frameworks (Google ADK 1.0, CrewAI, LangChain), A2A-Protokoll, RAG, Produktions-Deployment
